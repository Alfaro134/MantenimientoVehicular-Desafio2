# CarTrack UDB - Gestión de Mantenimiento Vehicular

[![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)
[![Expo](https://img.shields.io/badge/Expo-000020?style=for-the-badge&logo=expo&logoColor=white)](https://expo.dev/)

Desafío Práctico #2 para la asignatura de **Diseño y Programación de Software Multiplataforma (G01L)** en la Universidad Don Bosco. Esta aplicación permite a los usuarios llevar un control riguroso de las piezas sustituidas en sus vehículos, integrando métricas financieras de mantenimiento.

## Características Principal
- **Historial de Cambios:** Registro detallado de piezas, marcas, números de serie y precios.
- **Visualización por Fecha:** Listado automático ordenado cronológicamente para trazabilidad.
- **Dashboard de Gastos:** Pantalla de estadísticas con inversión total y gráficos por categoría.
- **UI Moderna:** Diseño basado en *Dark Neumorphism* para una experiencia de usuario premium.
- **Persistencia Local:** Almacenamiento de datos mediante `AsyncStorage`.

## Stack Tecnológico
* **Framework:** React Native (Expo SDK)
* **Lenguaje:** JavaScript / ES6+
* **Estilos:** StyleSheet (Native) con enfoque en sombras neumórficas.
* **Navegación:** React Navigation (Stack & Tabs).
* **Iconos:** Expo Vector Icons (Material Community, FontAwesome 5).
